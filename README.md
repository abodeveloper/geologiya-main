# GGI – veb-sayt

React + TypeScript + Vite asosidagi GGI instituti rasmiy veb-ilovasi.

## Texnologiyalar

- React 19, TypeScript, Vite
- React Router, TanStack Query, Zustand
- Tailwind CSS, Radix UI, Framer Motion
- i18next (uz / ru / en)

## Ishga tushirish

```bash
npm install
cp .env.example .env   # .env faylida VITE_API_URL o‘rnating
npm run dev
```

## Muhim o‘zgaruvchilar

- `VITE_API_URL` – backend API manzili (masalan: `https://api.example.com`)

## Skriptlar

- `npm run dev` – ishlab chiqish serveri
- `npm run build` – production build
- `npm run preview` – build’ni brauzerda ko‘rish
- `npm run lint` – ESLint
